# Named Entity Recognition (NER) using Hugging Face Pipeline

A Natural Language Processing (NLP) project that automatically identifies and classifies named entities from text using Hugging Face Transformers. The application extracts meaningful information such as people, organizations, locations, dates, and other entities from unstructured text.

---

## Overview

Named Entity Recognition (NER) is a fundamental NLP task that locates and classifies entities within text into predefined categories.

This project leverages pre-trained transformer models from Hugging Face to perform accurate entity extraction without requiring custom model training.

---

## Features

* Detect Person Names
* Identify Organizations
* Recognize Locations
* Extract Dates and Time Expressions
* Detect Miscellaneous Entities
* Fast inference using Hugging Face Pipeline
* Easy-to-use Python implementation
* Supports custom text inputs
